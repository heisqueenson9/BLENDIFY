## 🚀 HOW TO USE

### Option 1: Test Locally (3 steps)
1. Open Terminal in blendify folder
2. Run: `python -m http.server 3000`
3. Visit: http://localhost:3000

### Option 2: Deploy to Vercel (2 minutes)
1. Install Vercel: `npm install -g vercel`
2. Run: `vercel`
3. Follow prompts → Your site is live!
